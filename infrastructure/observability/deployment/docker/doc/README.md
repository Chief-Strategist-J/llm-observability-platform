# Docker Deployment Instructions

To run the observability service locally using Docker Compose, use the provided scripts in the `scripts/` directory.

## Pre-requisites
- Docker
- Docker Compose

## Execution
Run the following commands from the `docker/` directory:
- **Run Service**:
  ```bash
  ./scripts/run.sh
  ```
- **Build & Push Image**:
  ```bash
  ./scripts/build.sh
  ```
